## Projects

- **[Tech-News Sentiment Classifier](https://github.com/Karen2007/Tech-News-Sentiment)** Hourly pipeline that scrapes tech headlines, runs them through FinBERT, and aggregates a sentiment index. Automated end-to-end with GitHub Actions.
- **[Yerevan Air Quality Analysis](https://github.com/Karen2007/YerevanAirAnalysis)** Three regression models predicting PM2.5, PM10 and NO₂ in Yerevan from Open-Meteo weather data.
- **[BMW Sales Analysis](https://github.com/Karen2007/BMW-Sales-Analysis)** Exploratory analysis of used-car listings: MySQL-backed pipeline with aggregation queries and Python visualisation. The interesting part turned out to be the data quality issues that re-shaped the conclusions.
